# Design Brief: Forge Website

## Product Identity
- Type: Developer tool marketing site + SaaS landing
- Audience: Independent developers, solo founders, one-person company operators
- Tone: bold, developer-focused, modern

## Pages to Generate
| # | Page | Route | Purpose | Key Content |
|---|------|-------|---------|-------------|
| 1 | Homepage | / | Landing page — explain what Forge is and convert visitors | Hero, How it works, Pipeline viz, Terminal demo, Showcase preview, Competitor comparison, CTA |
| 2 | Showcase | /showcase | Display products built with Forge | Product cards with screenshots, stats, build logs |
| 3 | Pricing | /pricing | Show pricing tiers and convert to waitlist | 3 pricing cards, feature comparison, FAQ |
| 4 | Docs | /docs | Installation and usage guide | Step-by-step install, quick start, config, troubleshooting |
| 5 | About | /about | Project story and credibility | Creator story, vision, GitHub stats, contact |
